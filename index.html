핵심 요구사항 분석

  • 일반·기업 회원 통합 가입, 카카오·네이버·구글 소셜로그인, 본인 휴대폰 인증, 관심분야(부동산·주식·창업·부업) 기반 개인화
  • HLS 기반 VOD 스트리밍(배속·자막·화질·이어보기·구간 북마크)과 DRM 적용 다운로드 방지
  • 실시간 화상강의(Zoom API/YouTube 연동, 채팅·Q&A·화면공유·화이트보드·출석 자동 체크, LIVE 종료 후 VOD 자동 전환)
  • 강의 탐색·수강신청·PG 결제(카드/계좌이체/간편결제), 쿠폰·프로모션, 자동/수동 환불, 영수증·세금계산서 발행
  • 전문가 코칭 매칭(신청 폼·결제·코치 프로필·코칭 캘린더·분석 리포트), 세미나·스터디그룹·디지털 상품(E-Book) 판매
  • 학습 진도 관리·수료증 자동 생성/PDF, 시험(객관식·주관식·서술형)·평가 관리 및 통계
  • 커뮤니티(자료실·칼럼·수강후기·강사지원·FAQ·공지·이벤트·B2B 문의)와 통합 검색(자동완성·인기 검색어)
  • 관리자(강의·배너·회원·수강생·결제·통계 리포트)와 통합 알림(SMS·카카오알림톡·이메일) 발송
  • Spring Boot 3.3 + Java 21 LTS, MySQL/PostgreSQL, Redis, Docker 기반 CSAP 인증 클라우드 웹/DB 이중화 및 CI/CD

제안 포인트

본 프로젝트는 콘텐츠(VOD/라이브)·결제·코칭 매칭·커뮤니티·관리자가 유기적으로 연결된 대규모 통합 플랫폼입니다. 단일 기능 개발이 아니라 도메인 간 데이터 일관성과 권한 체계, 운영 안정성이 성패를 가르는 프로젝트로 판단하였으며, 아래 역량으로 대응합니다.

  • 대규모 통합 플랫폼 설계 역량 — 80~300개 규모의 API 엔드포인트와 다중 사용자 역할(학습자·기업·강사·코치·관리자)을 갖춘 B2B 포탈을 다수 설계·구축한 경험으로 도메인 모델과 권한(RBAC)을 견고하게 구성합니다.
  • 핀테크/재테크 도메인 이해 — VC 펀드 관리·P2P 투자 등 금융 플랫폼 수행 경험을 바탕으로 관심분야 기반 개인화, 투자 코칭 리포트, 정산·환불 흐름을 자연스럽게 설계합니다.
  • 결제·본인인증·외부 API 연동 실무 — PG 결제, 가상계좌, 본인 휴대폰 인증, 소셜로그인, 카카오알림톡/SMS 등 다수 외부 API 연동을 안정적으로 처리한 경험을 보유합니다.
  • VOD/DRM 및 실시간 화상 아키텍처 — HLS 적응형 스트리밍과 DRM 다운로드 방지, 외부 화상 솔루션(Zoom/YouTube) 연동 및 LIVE→VOD 전환 파이프라인을 설계합니다.
  • CSAP 클라우드 이중화 및 무중단 운영 — 웹/DB 이중화, Redis 캐시·세션, Docker 컨테이너, CI/CD 자동 배포 환경을 구성하여 가용성과 확장성을 확보합니다.
  • 자사 개발팀 4명과의 협업 체계 — 명확한 API 명세·브랜치 전략·이슈 트래킹을 기반으로 발주사 개발팀과의 병행 개발 및 지식 이관을 원활히 진행합니다.
참고: 상세 화면설계서(PPTX)·요구사항 정의서(XLSX) 첨부파일은 계약/NDA 이후 전달받아 화면 단위로 기능을 재정의하고 견적·일정을 정밀화할 것을 제안드립니다. 본 제안서는 공고 본문 기준으로 작성되었습니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

디지털 교육 수요 증가에 대응하여, 온라인 강의(VOD)부터 실시간 화상 코칭, 오프라인/온라인 세미나까지 학습의 전 과정을 하나의 플랫폼에서 통합 지원하는 것을 목표로 합니다. 기업 고객과 일반 학습자를 모두 수용하며, 부동산·주식·창업·부업 등 재테크 관심분야 기반의 개인화 학습 경험과 전문가 매칭 코칭을 핵심 차별점으로 제공합니다.

  • [필수] 회원정보 수정, 알림 수신(이메일/SMS/카카오톡) 설정, 회원 탈퇴

3.2 강의 탐색·결제

  • [필수] 그리드/리스트 뷰, 카테고리·난이도·가격 필터, 정렬, 상세페이지(커리큘럼·강사·후기·환불정책)
  • [필수] 수강신청/결제(카드·계좌이체·간편결제 PG), 쿠폰·프로모션, 자동/수동 환불, 영수증 발행

3.3 동영상·화상 강의

  • [필수] HLS 플레이어(배속·자막·화질), 진도 자동 저장·이어보기, 구간 북마크, 모바일 최적화
  • [필수] DRM 적용 다운로드 방지
  • [필수] 실시간 화상강의(Zoom API/YouTube), 채팅·Q&A, 화면공유·화이트보드, 출석 자동 체크, LIVE→VOD 자동 전환

3.4 학습관리·부가 기능

  • [필수] 진도율 대시보드·학습 이력, 이수 조건 충족 시 수료증 자동 생성, PDF 다운로드·이메일 발송
  • [필수] 전문가 코칭 매칭(신청 폼·결제, 코치 프로필, 코칭 캘린더, 분석 리포트)
  • [필수] 디지털 상품(PDF E-Book) 판매·뷰어, 세미나(온/오프) 생성·신청, 스터디 그룹

3.5 커뮤니티·평가

  • [필수] 자료실·칼럼·수강후기·강사지원·FAQ·공지사항·이벤트·B2B 문의 게시판
  • [필수] 시험(객관식·주관식·서술형) 생성·문제은행·자동/수동 채점, 평가지 생성·통계 보고서

3.6 관리자

  • [필수] 강의·배너·팝업 관리, 강의 상태 관리, 카테고리·태그·강사 배정
  • [필수] 회원·수강생 관리(검색·필터·등급), 수료 처리, 결제·환불 내역·세금계산서
  • [필수] 매출·수강·강의 통계 대시보드, 리포트 엑셀 다운로드, 통합 알림(SMS·카카오알림톡·이메일)
